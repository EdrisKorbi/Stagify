# Stagify 🚀

Plateforme de mise en relation entre étudiants et entreprises pour trouver des stages.

## ✨ Fonctionnalités

- Authentification (Étudiant / Entreprise)
- Recherche de stages par filière
- Interface moderne avec animations
- Système de candidature
- Quiz de compatibilité

## 🛠️ Technologies utilisées

- HTML
- CSS
- JavaScript

## 📁 Structure du projet
project/
│── index.html
│── css/
│── js/

## 🚀 Lancer le projet

Ouvrir simplement `index.html` dans le navigateur.
